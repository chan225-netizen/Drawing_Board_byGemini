## Drawing_Board_byGemini
手势识别的“空中画布”程序

## 功能描述

这段代码是通过Gemini的实现了一个基于摄像头和 MediaPipe 手势识别的“空中画布”程序。

它会打开摄像头画面，检测手部关键点：当单手拇指和食指捏合时，就把捏合点当作画笔，在画面上绘制彩虹渐变颜色的线条；松开手指则停止绘制。检测到双手时，会在两只手相关区域画出彩色框。程序还支持快捷键操作：C 清空画布，L 开关灰度滤镜，W 切换全屏，V 录制视频并保存，Q 退出。

整体作用：用手势在摄像头画面上进行实时“隔空绘画”，并提供滤镜、录制、清屏和全屏功能。

## 运行环境

Python 3.12.8

### 依赖库

需要安装以下 Python 库：

opencv-python mediapipe numpy

测试版本：opencv-python: 4.11.0 mediapipe: 0.10.21 numpy: 1.26.4
